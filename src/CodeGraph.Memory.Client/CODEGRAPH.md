## Summary

CodeGraph.Memory.Client is a .NET HTTP client library that provides typed access to a remote memory graph service. It exposes operations for querying, traversing, writing, and managing a knowledge graph of entities and claims on behalf of named users, communicating over HTTP with JSON serialization. The project defines the client interface, its HTTP implementation, configuration options, and a DI registration extension for ASP.NET Core applications.
