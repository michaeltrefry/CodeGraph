## Summary

CodeGraph.Indexer.Host is an ASP.NET Core web application that serves as the hosting layer for a code graph indexing system. It exposes HTTP API endpoints for triggering indexing operations (repository processing, schema sync, community detection, cross-repo linking, batch analysis) and hosts MassTransit message consumers that react to domain events such as repository processing, analysis completion, and removal. The project orchestrates the full indexing pipeline by coordinating between a graph store, project/analysis services, and a TypeScript sidecar process.
