## Summary

CodeGraph.Indexer.Host is an ASP.NET Core web application that serves as the hosting entry point for a code graph indexing system. It exposes HTTP API endpoints for triggering repository indexing operations (discovery, processing, schema sync, cross-repo linking, community detection, batch analysis) and wires up MassTransit message consumers that react to indexing lifecycle events. The project coordinates between a graph store, project/batch analysis services, and a TypeScript sidecar process to build and maintain a code knowledge graph.
