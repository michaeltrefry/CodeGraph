## Summary

CodeGraph.Indexer.Host is an ASP.NET Core web application that serves as the host process for a code graph indexing pipeline. It exposes HTTP endpoints for triggering indexing operations (repository processing, schema synchronization, linking, community detection, batch analysis) and wires up a set of MassTransit message consumers that react to pipeline events such as repository indexing completion, analysis batch submission, and repository removal. The project orchestrates the full lifecycle of building and maintaining a graph representation of code repositories.
