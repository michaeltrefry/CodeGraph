## Summary

CodeGraph.Indexer.Host is an ASP.NET Core web host that orchestrates code repository indexing and graph construction. It exposes a REST API for triggering indexing operations (repository processing, schema sync, cross-repo linking, community detection, batch analysis) and uses MassTransit message consumers to coordinate an event-driven pipeline that processes repositories, stores results in a graph store, and performs post-indexing analysis. It also manages a TypeScript sidecar process used during analysis.
