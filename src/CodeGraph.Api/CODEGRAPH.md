## Summary

CodeGraph.Api is an ASP.NET Core Web API that serves as the HTTP and messaging entry point for the CodeGraph system — a platform for indexing, analyzing, and querying source code repositories as a graph. It exposes REST endpoints for querying code nodes, projects, clusters, reviews, memory/knowledge graphs, wiki content, and administrative settings, while also hosting MassTransit message consumers that react to background pipeline events such as repository indexing, batch analysis, and synthesis completion.
