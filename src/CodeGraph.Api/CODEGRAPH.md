CodeGraph.Api is the ASP.NET Core Web API layer for the CodeGraph platform — a code intelligence system that indexes source repositories, builds a semantic graph of code nodes, runs LLM-based analysis/review, maintains a memory graph, and exposes all of this through a REST API.
