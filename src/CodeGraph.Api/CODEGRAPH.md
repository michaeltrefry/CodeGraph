## Summary

CodeGraph.Api is an ASP.NET Core Web API project that serves as the primary HTTP gateway for the CodeGraph platform — a system that indexes, analyzes, and graphs code repositories. It exposes REST endpoints for querying code nodes, projects, clusters, schemas, search, AI assistant interactions, memory graphs, wiki content, and administrative operations, while also acting as a message bus consumer that reacts to indexing, analysis, and repository lifecycle events published by other services.
