## Summary

CodeGraph.Api is an ASP.NET Core Web API that serves as the central orchestration and HTTP interface layer for the CodeGraph system — a platform that indexes, analyzes, and queries source code repositories as a graph. It exposes REST endpoints for querying code graph nodes, projects, clusters, memory, wiki, search, and AI-assisted code review, while also hosting MassTransit message consumers that react to asynchronous events such as repository indexing completion, analysis batch progress, and memory claim storage.
