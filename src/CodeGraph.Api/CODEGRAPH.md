## Summary

CodeGraph.Api is an ASP.NET Core Web API that serves as the backend for a code analysis and knowledge graph platform. It exposes REST endpoints for querying and managing a graph of code nodes (projects, files, methods, etc.), running repository indexing and analysis pipelines, managing a wiki and memory store, and interacting with an AI assistant. It also hosts a set of MassTransit message consumers that react to asynchronous pipeline events such as repository indexing completion, batch analysis submission, and analysis synthesis completion.
