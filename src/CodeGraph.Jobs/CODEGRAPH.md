## Summary

CodeGraph.Jobs is a background job scheduling and execution service for the CodeGraph system. It defines a set of named job types (discover repositories, re-index, batch analysis, link/detect, community detection, MCP doc regeneration), dispatches them by name with JSON-serialized arguments, and runs them on cron-based or on-demand schedules managed through a persistent store. The project hosts a .NET Generic Host worker process that polls for due schedules and executes them, while also exposing service interfaces for external callers to manage schedules via CRUD operations.
