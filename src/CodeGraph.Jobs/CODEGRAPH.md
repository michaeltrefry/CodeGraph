## Summary

CodeGraph.Jobs is an ASP.NET Core web service that manages and executes background jobs for analyzing code repositories. It provides HTTP endpoints for triggering jobs (discover repositories, process repositories, process batch results), a cron-based scheduling system for automated job execution, and a dispatcher layer that routes named job types to their corresponding implementations. The project serves as the job orchestration backbone of the CodeGraph repository analysis platform.
