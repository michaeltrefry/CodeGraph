## Summary

CodeGraph.Tests is the primary test project for the CodeGraph repository, providing unit and integration tests across all major subsystems including extractors, data stores, services, API controllers, memory systems, and host startup configurations. It contains a large collection of in-memory fakes, recording stubs, and no-op implementations of nearly every service interface in the system, enabling isolated unit testing without real database or external service dependencies.
