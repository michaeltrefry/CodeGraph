## Summary

CodeGraph.Tests is the automated test project for the CodeGraph repository, containing unit and integration tests that verify the behavior of services, controllers, data stores, extractors, memory subsystems, and infrastructure components. It heavily uses hand-written test doubles (fakes, recording implementations, stubs, and no-op implementations) of core interfaces rather than a mocking framework, and provides a comprehensive InMemoryGraphStore that implements the full IGraphStore contract in memory to support fast, hermetic service-level tests. The project spans controller tests, service logic tests, MariaDB integration tests, extractor tests (Roslyn, SQL, Terraform, Ansible, ColdFusion, etc.), and startup/DI registration validation tests.
