## Summary

CodeGraph.Metrics is an ASP.NET Core microservice responsible for collecting and persisting telemetry/metrics events within the CodeGraph system. It subscribes to domain messages (LLM usage and MCP tool invocations) via MassTransit message broker, processes them through a metrics event recorder, and stores results in a configured database (supporting MariaDB). The project includes persistence registration with migration support, internal service authentication middleware, and standard consumer retry configuration.
