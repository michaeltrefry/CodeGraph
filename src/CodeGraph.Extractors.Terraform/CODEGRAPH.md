## Summary

CodeGraph.Extractors.Terraform is a code analysis plugin that parses HashiCorp Configuration Language (HCL) files used by Terraform infrastructure-as-code. It extracts structural elements such as resources, modules, data sources, variables, outputs, and locals from .tf and .tfvars files, then produces a graph of nodes and edges representing relationships (references, dependencies) between infrastructure components. This project implements the ICodeExtractor interface, making it a pluggable extractor within the larger CodeGraph repository's extraction pipeline.
