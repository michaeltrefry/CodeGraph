## Summary

CodeGraph.Memory.Host is an ASP.NET Core web API host that exposes a memory graph service. It provides REST endpoints for storing, querying, searching, and managing a knowledge graph of memory claims and entities, and also consumes messages from a MassTransit message bus to asynchronously ingest memory claims. The project handles persistence registration, authentication middleware, and health probing for the write pipeline.
